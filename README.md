# projetointegrado1
Esse projeto feito por Arthur Yuzo e Pedro Brasil da turma de desenvolvimento de sistemas.

Nosso projeto se inicia com o botão 9 pressionado, a aula inicia e uma tela de bem-vindo é mostrada, após isso nosso programa cria uma senha aleatória, e depois verifica se a senha que for digitada foi bloqueada se a senha digitada for diferente da senha aleatória 3 vezes ela é bloqueada, caso ela fosse bloqueada o display mostraria que foi bloqueada e deixa sem mexer por 10s, depois ele entra dentro de um while para sair só quando a senha for correta e não estiver bloqueada, depois o botão 12 aumenta o número do primeiro caractere da senha, o caractere é confirmado após o botão 11 ser pressionado, isso se repete para os próximos 3 caracteres para assim formar uma senha de 4 dígitos, depois todos os caracteres são multiplicados para serem armazenados em uma int de 4 algarismos, depois é verificado se a senha digitada é incorreta, se isso acontecer aumenta a quantidade de tentativas e se essas tentativas ultrapassarem 3 a senha é bloqueada, se a senha for correta é mostrado no display que a senha foi correta, depois é pedido para digitar a quantidade de alunos máxima, se o botão 12 for pressionado ele aumenta a quantidade de dezenas de alunos máxima na sala, se o botão 11 for pressionado é necessário digitar a unidade, (todas as vezes que um usuário necessitar de digitar algo o botão 10 pode ser usado para diminuir caso o botão 11 seja pressionado errado), após digitar a quantidade máxima, é usado um while para que assim caso o botão 11 e 12 não sejam pressionados mostrar a quantidade de alunos presentes (aumenta quando clicar no botão 9), quantidade de alunos fora da sala (aumenta quando clicar no botão 11), e uma barra comparando a quantidade de alunos presentes na sala com a quantidade total de alunos que podem ter dentro da sala, caso a quantidade de alunos fora de sala seja igual a 3 e o botão 11 for pressionado aparece um aviso mostrando que o limite já foi atingido e que não podem sair mais alunos, e caso algum aluno volte do banheiro é necessário apertar o botão 12, agora caso o botão 11 e o botão 12 forem pressionados simultaneamente a aula acaba mostrando um relatório de quantidades totais de alunos que saíram de sala e a quantidade total de alunos que estiveram presentes dentro da sala de aula.
